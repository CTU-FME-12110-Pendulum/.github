# CTU FME 12110 - Rotary Inverted Pendulum

A rotary inverted pendulum developed at the Departement of Instrumentation and Control of the Faculty of Mechanical Engineering at CTU. The main purpose of this project is to provide a robust benchmarking platform for testing nonlinear model-based control approaches (trajectory optimization, state estimation, parameter identification).
